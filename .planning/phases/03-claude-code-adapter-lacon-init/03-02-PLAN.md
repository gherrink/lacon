<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User command string → DFA | Adversarial input includes nested constructs, malicious heredocs, chain ops inside opaque regions. Splitter must never miscount or panic. |
| Splitter output → Plan 4 reassembly | Plan 4 reassembles by joining segment.text + trailing_op_span. If the span fields are wrong, an injection vector opens (T-injection-chain-reassembly). |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-01 | T (Tampering) | DFA state machine | mitigate | Byte-iterating with explicit state for every opaque region (single-quote, double-quote, subshell, cmd-sub, backtick, process-sub, heredoc). All 13 scenarios + 2 pathological-input tests exercise the state machine. Compromise mode would be a chain op inside an opaque region splitting incorrectly — every scenario S7–S11 is the exact regression test. |
| T-03-02-02 | T (Tampering) | Reassembly via trailing_op_span | mitigate | Each scenario test asserts `segments[i].text + segments[i].trailing_op_span.unwrap_or_default() + ...` reproduces the original input byte-exact. Plan 4 reassembly relies on this property. |
| T-03-02-03 | D (Denial of service) | Pathological inputs | mitigate | DFA is O(n) single-pass; no recursive descent; no regex backtracking. Pathological tests in chain_split.rs (nested subshells, deeply-nested cmd-sub) exercise the DFA's linearity. |
| T-03-02-04 | I (Information disclosure) | Heredoc body | mitigate | Heredoc bodies are opaque — chain operators inside the body do NOT split. The fixture `cat <<EOF\na && b\nEOF` → 1 segment locks this. The 03-RESEARCH.md:534 fallback (opaque-until-EOL) is acceptable per Claude's-discretion if full heredoc tracking proves brittle. |
| T-03-02-05 | T (Tampering) | Pipeline-as-segment | mitigate | S10 (`a | b && c` → 2 segments) locks REQ-adapter-pipes-passthrough. `|` is never a split point (D-09). Test asserts segment.text contains the pipe. |
