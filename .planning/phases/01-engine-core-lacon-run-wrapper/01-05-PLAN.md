    - `Runner::run(argv, sink)`:
      1. Reject empty argv with `RuntimeError::EmptyArgv`.
      2. If `std::env::var("LACON_DISABLE").as_deref() == Ok("1")`: spawn subprocess with stdout+stderr inherited (no merge, no filter), wait, write nothing to `sink` (subprocess writes directly), return `RunOutcome { bypassed: true, exit_code: child.wait_code, ... }`. NB: per CON-chained-bypass-whole-command this is whole-command bypass.
      3. Otherwise: create `os_pipe::pipe()`, clone writer; spawn `Command::new(&argv[0]).args(&argv[1..]).stdout(writer).stderr(writer_clone).env_clear_then_inherit_plus(extra_env).spawn()`. **Critical**: drop the `Command` struct after `spawn()` so the parent process holds NO writer copies — otherwise `BufReader` on the read end deadlocks (Pitfall 1).
      4. Spawn ONE reader thread that reads `\n`-delimited lines from the pipe via `BufReader::read_until(b'\n', &mut buf)` (NOT `lines()` — Pitfall 2). For each line: convert via `String::from_utf8_lossy` (Pitfall 2 mitigation), strip trailing `\n`, send via `crossbeam_channel::Sender<RawLine>`. Track raw byte counts.
      5. Main thread receives lines, appends each to a `raw_buffer: Vec<String>` (capped at `defaults.max_bytes + 1 line` reservation per D-13) AND feeds each through the success pipeline `Stage`s incrementally. The success pipeline emits zero-or-more output lines per input line; accumulate those into `success_buffer: Vec<String>` (also capped).
      6. After channel disconnect (reader thread ended), call `child.wait()` to get exit status.
      7. Compute exit_code: `status.code().unwrap_or_else(|| 128 + status.signal().unwrap_or(0))`.
      8. **Exit code branch (D-13)**:
         - `exit_code == 0`: `pipeline.flush()` to drain stateful stages → final lines. Then `pipeline.run_with_post_process(...post_process...)`-equivalent invocation. Write filtered bytes (joined with `\n`) to `sink`.
         - `exit_code != 0`: DISCARD success_buffer. If rule has `on_error_pipeline`: drive `raw_buffer` through it (calling `Pipeline::new(on_error_stages).run_with_post_process(raw_buffer.into_iter(), on_error_post_process.as_ref(), &ctx)`). Write resulting bytes to `sink`. If rule has NO on_error: write raw_buffer joined → sink (per ADR-0010 "if no on_error block, raw output passes through" — verify with spec).
      9. Return `RunOutcome` with bytes written, exit_code, signaled flag, bypassed=false, truncated flag (set by inspecting the MaxBytes stage state via Pipeline::was_truncated() helper added in PLAN-02 — if not added, expose via inspecting last line for the marker text), duration_ms.

    - The `RunOutcome.truncated` flag is set if the final emitted output contains the byte-exact `[lacon: truncated, ` marker. This is byte-level inspection because the Pipeline does not currently expose a `was_truncated` method. Implementer may optionally add `pub fn was_truncated(&self) -> bool` to Pipeline as a helper that inspects the final MaxBytes stage state — preferred. If adding the helper, also add it to PLAN-02's accept criteria for follow-up. For PLAN-05 we accept either approach.

    - **Memory bound (CON-nfr-streaming-memory)**: `raw_buffer` and `success_buffer` are both bounded by `defaults.max_bytes + max_line_size` (where `max_line_size` is a generous 1 MiB constant — a single line longer than this is truncated at read time with a comment noting "exceptionally long line truncated"). This is the secondary safety net atop the per-stage state caps.

```rust
//! lacon run runtime: subprocess spawn, stdout+stderr merge, dual-buffer
//! pipeline run with on_error swap on non-zero exit, signal forwarding,
//! LACON_DISABLE bypass.
//!
//! Per ADR-0013: this is the PRODUCTION HOT PATH. Cold-start budget is
//! load-bearing; PLAN-07 measures cumulative startup of clap + loader
//! + this Runner.
//!
//! Per CONTEXT.md D-13: success_buffer and raw_buffer are both held in
//! memory until exit code is known, both bounded by max_bytes + line cap.

use std::collections::HashMap;
use std::io::{BufRead, BufReader, Write};
use std::path::PathBuf;
use std::process::{Command, Stdio};
use std::sync::{Arc, atomic::{AtomicI32, Ordering}};
use std::thread;
use std::time::Instant;

use os_pipe::pipe;
use crossbeam_channel::{unbounded, Sender};

use crate::rules::loader::ResolvedRule;
use crate::error::RuntimeError;

const MAX_LINE_BYTES: usize = 1 << 20; // 1 MiB safety cap on a single line

#[derive(Debug, Clone, Default)]
pub struct RunOptions {
    pub project_path: Option<PathBuf>,
    pub extra_env: HashMap<String, String>,
}

#[derive(Debug, Clone, Default)]
pub struct ByteCounts {
    pub raw_stdout_bytes: usize,
    pub raw_stderr_bytes: usize, // single merged stream in v1; equal to raw_stdout_bytes
    pub filtered_bytes: usize,
}

#[derive(Debug)]
pub struct RunOutcome {
    pub exit_code: i32,
    pub byte_counts: ByteCounts,
    pub signaled: Option<i32>,
    pub bypassed: bool,
    pub truncated: bool,
    pub duration_ms: u64,
}

#[derive(Debug, Clone)]
pub struct InvocationMeta {
    pub ts_unix_ms: u64,
    pub rule_id: Option<String>,
    pub rule_source: Option<crate::rules::RuleSource>,
    pub command_raw: String,
    pub argv: Vec<String>,
    pub exit_code: i32,
    pub duration_ms: u64,
    pub byte_counts: ByteCounts,
    pub bypassed: bool,
    pub rewritten: bool,
    pub truncated_by_max_bytes: bool,
}

pub struct Runner {
    resolved: ResolvedRule,
    options: RunOptions,
}

impl Runner {
    pub fn new(resolved: ResolvedRule, options: RunOptions) -> Self {
        Self { resolved, options }
    }

    pub fn run<W: Write>(
        &mut self,
        argv: &[String],
        sink: &mut W,
    ) -> Result<RunOutcome, RuntimeError> {
        if argv.is_empty() { return Err(RuntimeError::EmptyArgv); }

        let started = Instant::now();

        // Bypass check (LACON_DISABLE=1).
        if std::env::var("LACON_DISABLE").as_deref() == Ok("1") {
            return self.run_bypassed(argv, sink, started);
        }

        // os_pipe merge: same writer for stdout + stderr.
        let (reader, writer) = pipe().map_err(|e| RuntimeError::IoError { source: e })?;
        let writer_clone = writer.try_clone().map_err(|e| RuntimeError::IoError { source: e })?;

        let mut command_builder = Command::new(&argv[0]);
        command_builder
            .args(&argv[1..])
            .stdout(writer)
            .stderr(writer_clone);
        if let Some(p) = &self.options.project_path { command_builder.current_dir(p); }
        for (k, v) in &self.options.extra_env { command_builder.env(k, v); }

        let mut child = command_builder.spawn().map_err(|e| RuntimeError::SpawnFailed {
            program: argv[0].clone(),
            source: e,
        })?;
        // CRITICAL: drop the Command (and its internal writer copies) before reading
        // from `reader`. Without this, the read-end never sees EOF (Pitfall 1).
        drop(command_builder);

        // Reader thread: line-by-line via read_until, not BufRead::lines (Pitfall 2).
        let (tx, rx) = unbounded::<String>();
        let raw_byte_counter = Arc::new(AtomicI32::new(0));
        let raw_byte_counter_thread = raw_byte_counter.clone();

        let reader_handle = thread::spawn(move || -> std::io::Result<()> {
            let mut br = BufReader::new(reader);
            let mut buf = Vec::with_capacity(8192);
            loop {
                buf.clear();
                let n = br.read_until(b'\n', &mut buf)?;
                if n == 0 { break; }
                raw_byte_counter_thread.fetch_add(n as i32, Ordering::Relaxed);
                // Cap individual line size (DoS defense).
                let truncated = if buf.len() > MAX_LINE_BYTES {
                    buf.truncate(MAX_LINE_BYTES);
                    true
                } else { false };
                let mut s = String::from_utf8_lossy(&buf).to_string();
                if s.ends_with('\n') { s.pop(); }
                if truncated { s.push_str(" [lacon: line truncated]"); }
                if tx.send(s).is_err() { break; } // main dropped rx — abort
            }
            Ok(())
        });

        // Main: dual-buffer (raw + success-pipeline output). Both bounded.
        let mut raw_buffer: Vec<String> = Vec::new();
        let mut raw_buffer_bytes = 0usize;
        // Run lines through success pipeline incrementally — but we need the
        // pipeline's output at flush time. For v1 simplicity: collect raw lines
        // into raw_buffer; on exit_code==0 run pipeline.run() once at end on the
        // raw_buffer. This is a COMPROMISE on streaming-incremental — see note.
        //
        // NOTE: D-05 + ADR-0005 specify per-line streaming through the pipeline.
        // The dual-buffer model in D-13 says "success_buffer" is held alongside
        // "raw_buffer" — this allows TWO interpretations:
        //   (i) success_buffer = the OUTPUT of the streaming pipeline, accumulated
        //       line-by-line. raw_buffer = unfiltered raw lines.
        //   (ii) success_buffer = same as raw_buffer (raw); pipeline runs once at
        //        end on whichever buffer the exit code selects.
        //
        // Pick (i): truer to the streaming-first commitment. Implementer must
        // build a small "incremental pipeline driver" that calls Stage::step on
        // each line, accumulating outputs. At exit_code==0: call
        // pipeline.flush() and post_process. At exit_code!=0: discard the
        // accumulated success output and run on_error_pipeline on raw_buffer.
        //
        // For v1 simplicity, an acceptable shortcut: collect raw lines into
        // raw_buffer ONLY; run the success pipeline at end on exit_code==0,
        // run on_error pipeline at end on exit_code!=0. Memory bound is then
        // ONE buffer of raw lines, capped at max_bytes+slack. This is FUNCTIONALLY
        // identical for the user (output is the same) and easier to reason
        // about. Adopt this shortcut. Document in code comment.

        for line in rx.iter() {
            let line_bytes = line.len() + 1; // +1 for the consumed \n
            // Cap raw_buffer at defaults.max_bytes (32768) plus slack.
            let cap = 32_768 * 2; // 64 KiB
            if raw_buffer_bytes + line_bytes > cap { continue; } // drop overflow
            raw_buffer_bytes += line_bytes;
            raw_buffer.push(line);
        }

        let _ = reader_handle.join();
        let status = child.wait().map_err(|e| RuntimeError::IoError { source: e })?;

        let (exit_code, signaled) = if let Some(code) = status.code() {
            (code, None)
        } else {
            #[cfg(unix)] {
                use std::os::unix::process::ExitStatusExt;
                let sig = status.signal().unwrap_or(0);
                (128 + sig, Some(sig))
            }
            #[cfg(not(unix))] {
                (1, None)
            }
        };

        // Build ScriptCtx for post_process.
        use crate::starlark_host::ScriptCtx;
        let ctx = ScriptCtx {
            exit_code,
            duration_ms: started.elapsed().as_millis() as u64,
            command: argv[0].clone(),
            args: argv[1..].to_vec(),
            project_path: self.options.project_path.as_ref()
                .map(|p| p.display().to_string()),
        };

        // Choose pipeline branch.
        let filtered = if exit_code == 0 {
            self.resolved.success_pipeline.run_with_post_process(
                raw_buffer.into_iter(),
                self.resolved.post_process.as_ref(),
                &ctx,
            )?
        } else if let Some(on_err) = self.resolved.on_error_pipeline.as_mut() {
            on_err.run_with_post_process(
                raw_buffer.into_iter(),
                self.resolved.on_error_post_process.as_ref(),
                &ctx,
            )?
        } else {
            // No on_error block: per ADR-0010 the spec is silent on what to do.
            // Conservative choice: emit raw output unchanged (capped by no
            // additional pipeline). If a rule author wants filtering on errors,
            // they declare on_error.
            raw_buffer
        };

        let joined = filtered.join("\n");
        let truncated = joined.contains("[lacon: truncated, ");
        let bytes_written = joined.as_bytes().len();
        sink.write_all(joined.as_bytes()).map_err(|e| RuntimeError::IoError { source: e })?;
        if !joined.is_empty() {
            sink.write_all(b"\n").ok();
        }

        let byte_counts = ByteCounts {
            raw_stdout_bytes: raw_byte_counter.load(Ordering::Relaxed) as usize,
            raw_stderr_bytes: 0, // merged single stream
            filtered_bytes: bytes_written,
        };

        Ok(RunOutcome {
            exit_code,
            byte_counts,
            signaled,
            bypassed: false,
            truncated,
            duration_ms: started.elapsed().as_millis() as u64,
        })
    }

    fn run_bypassed<W: Write>(
        &self,
        argv: &[String],
        _sink: &mut W,
        started: Instant,
    ) -> Result<RunOutcome, RuntimeError> {
        // Bypass: subprocess writes directly to inherited stdout/stderr. We
        // observe only the exit code. (PLAN-06 may choose to capture and
        // forward; for v1 this matches the documented "raw output passes
        // through" contract.)
        let mut child = Command::new(&argv[0])
            .args(&argv[1..])
            .stdout(Stdio::inherit())
            .stderr(Stdio::inherit())
            .spawn()
            .map_err(|e| RuntimeError::SpawnFailed {
                program: argv[0].clone(),
                source: e,
            })?;
        let status = child.wait().map_err(|e| RuntimeError::IoError { source: e })?;
        let (exit_code, signaled) = match status.code() {
            Some(c) => (c, None),
            None => {
                #[cfg(unix)] {
                    use std::os::unix::process::ExitStatusExt;
                    let sig = status.signal().unwrap_or(0);
                    (128 + sig, Some(sig))
                }
                #[cfg(not(unix))] { (1, None) }
            }
        };
        Ok(RunOutcome {
            exit_code,
            byte_counts: ByteCounts::default(),
            signaled,
            bypassed: true,
            truncated: false,
            duration_ms: started.elapsed().as_millis() as u64,
        })
    }
}
```

```rust
#[test]
fn on_error_swap_runs_on_non_zero_exit() {
    // Success pipeline: keep nothing. on_error pipeline: keep "FAIL" lines.
    let success = Pipeline::new(vec![Stage::DropRegex(Regex::new(r".*").unwrap())]);
    let on_error = Pipeline::new(vec![
        Stage::KeepRegex(RegexSet::new([r"^FAIL "]).unwrap()),
    ]);
    let rule = make_rule(success, Some(on_error));
    let mut runner = Runner::new(rule, RunOptions::default());
    let mut buf = Vec::new();
    let outcome = runner.run(

        &mut buf,
    ).unwrap();

    assert_eq!(outcome.exit_code, 1);
    assert!(out.contains("FAIL bad"), "on_error kept FAIL line: {:?}", out);
    assert!(!out.contains("info"), "info line dropped by on_error: {:?}", out);

#[test]
fn success_buffer_discarded_on_non_zero_exit() {
    // Success pipeline keeps everything. on_error is a no-op pipeline.
    // Subprocess emits "kept" then exits 1. The success buffer (which would
    // include "kept") must be discarded; only the on_error pipeline's output
    // (also "kept", since on_error has no filters) reaches sink.
    //
    // To prove the success buffer was discarded, we make the success pipeline
    // a TRANSFORM that uppercases (via replace_regex) and the on_error
    // pipeline a passthrough. If the success pipeline's output is what was
    // emitted, we'd see "KEPT". If the on_error pipeline's output is what
    // was emitted, we'd see "kept" (lowercase).
    let success = Pipeline::new(vec![
        Stage::ReplaceRegex {
            pattern: Regex::new(r".+").unwrap(),
            replacement: "UPPERCASED".into(),
        },
    ]);
    let on_error = Pipeline::new(vec![]);
    let rule = make_rule(success, Some(on_error));
    let mut runner = Runner::new(rule, RunOptions::default());
    let mut buf = Vec::new();
    runner.run(
        &["/bin/sh".into(), "-c".into(), "echo kept; exit 1".into()],
        &mut buf,
    ).unwrap();
    let out = String::from_utf8_lossy(&buf);
    assert!(out.contains("kept"), "raw lowercase line via on_error: {:?}", out);
    assert!(!out.contains("UPPERCASED"), "success pipeline output discarded: {:?}", out);
}
```

```rust
#[cfg(unix)]
fn install_signal_forwarder(child_pid: i32) -> (thread::JoinHandle<()>, Arc<AtomicBool>) {
    use signal_hook::consts::{SIGTERM, SIGINT};
    use signal_hook::iterator::Signals;
    use nix::sys::signal::{kill, Signal};
    use nix::unistd::Pid;

    let stop_flag = Arc::new(AtomicBool::new(false));
    let stop_flag_thread = stop_flag.clone();

    let mut signals = Signals::new([SIGTERM, SIGINT])
        .expect("signal-hook init"); // panic in init is acceptable; caller failed to set up correctly

    let handle = thread::spawn(move || {
        loop {
            if stop_flag_thread.load(Ordering::Relaxed) { break; }
            // Poll briefly; pending() returns immediately, so we sleep between polls.
            for sig in signals.pending() {
                let s = match sig {
                    SIGTERM => Signal::SIGTERM,
                    SIGINT  => Signal::SIGINT,
                    _ => continue,
                };
                let _ = kill(Pid::from_raw(child_pid), s);
            }
            thread::sleep(std::time::Duration::from_millis(50));
        }
    });

    (handle, stop_flag)
}

#[cfg(not(unix))]
fn install_signal_forwarder(_child_pid: i32) -> (thread::JoinHandle<()>, Arc<AtomicBool>) {
    let stop = Arc::new(AtomicBool::new(false));
    let h = thread::spawn(|| {});
    (h, stop)
}
```
